# HTML/JavaScript Integration

The simplest way to integrate AdUnblock into your website is using our JavaScript snippet. This method works with any website and requires no additional dependencies.

## Quick Installation

<Steps>

## Basic Implementation

Add this script tag to the `<head>` section of your HTML:

```html
<script>
  (function() {
    var script = document.createElement('script');
    script.src = 'https://cdn.ad-unblock.com/adunblock.min.js';
    script.async = true;
    script.onload = function() {
      AdUnblock.init({
        apiKey: 'YOUR_API_KEY',
        // Additional configuration options
      });
    };
    document.head.appendChild(script);
  })();
</script>
```

<Warning>
  Replace `YOUR_API_KEY` with your actual API key from the AdUnblock dashboard.
</Warning>

## Configuration Options

Customize AdUnblock's behavior with these configuration options:

```javascript
AdUnblock.init({
  apiKey: 'YOUR_API_KEY',
  
  // Detection settings
  detectDelay: 1000,        // Delay before detection (ms)
  detectRetries: 3,         // Number of detection attempts
  
  // Display options
  showMessage: true,        // Show detection message
  messageType: 'overlay',   // 'overlay', 'banner', 'redirect'
  messageTheme: 'light',    // 'light', 'dark', 'custom'
  
  // Behavior settings
  allowClose: true,         // Allow users to close message
  respectWhitelist: true,   // Respect user whitelist choices
  
  // Custom messages
  title: 'Ad Blocker Detected',
  message: 'Please consider disabling your ad blocker to support our content.',
  buttonText: 'I\'ve disabled my ad blocker',
  
  // Callbacks
  onDetected: function() {
    console.log('Ad blocker detected');
  },
  onWhitelisted: function() {
    console.log('Site whitelisted');
  },
  onClosed: function() {
    console.log('Message closed');
  }
});
```

## Message Types

### Overlay Message

The default overlay appears on top of your content:

```javascript
AdUnblock.init({
  apiKey: 'YOUR_API_KEY',
  messageType: 'overlay',
  messageTheme: 'light'
});
```

### Banner Message

A less intrusive banner at the top or bottom of the page:

```javascript
AdUnblock.init({
  apiKey: 'YOUR_API_KEY',
  messageType: 'banner',
  bannerPosition: 'top' // 'top' or 'bottom'
});
```

### Redirect

Redirect users to a dedicated page:

```javascript
AdUnblock.init({
  apiKey: 'YOUR_API_KEY',
  messageType: 'redirect',
  redirectUrl: '/adblock-detected'
});
```

## Advanced Features

### Conditional Loading

Load AdUnblock only on specific pages:

```javascript
// Only load on blog posts
if (window.location.pathname.includes('/blog/')) {
  AdUnblock.init({
    apiKey: 'YOUR_API_KEY'
  });
}
```

### Custom Detection Logic

Add your own detection logic:

```javascript
AdUnblock.init({
  apiKey: 'YOUR_API_KEY',
  customDetection: function() {
    // Your custom ad blocker detection code
    return isAdBlockerDetected;
  }
});
```

### Analytics Integration

Track ad blocker events in Google Analytics:

```javascript
AdUnblock.init({
  apiKey: 'YOUR_API_KEY',
  onDetected: function() {
    gtag('event', 'adblock_detected', {
      event_category: 'AdBlock',
      event_label: 'User has ad blocker'
    });
  },
  onWhitelisted: function() {
    gtag('event', 'adblock_whitelisted', {
      event_category: 'AdBlock',
      event_label: 'User whitelisted site'
    });
  }
});
```

## Troubleshooting

### Common Issues

**Script not loading:**
- Check your API key is correct
- Verify the script URL is accessible
- Check browser console for errors

**Detection not working:**
- Ensure the script loads after the DOM is ready
- Check if other scripts are interfering
- Verify ad blocker is actually active

**Message not appearing:**
- Check CSS conflicts
- Verify message type configuration
- Test with different ad blockers

### Performance Optimization

```javascript
// Lazy load AdUnblock
window.addEventListener('load', function() {
  AdUnblock.init({
    apiKey: 'YOUR_API_KEY'
  });
});

// Or use intersection observer for specific elements
const observer = new IntersectionObserver(function(entries) {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      AdUnblock.init({
        apiKey: 'YOUR_API_KEY'
      });
      observer.disconnect();
    }
  });
});
```
